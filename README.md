# Gym Coach

Персональный AI-тренер: программа Full Body A/B/C, техника упражнений, прогрессивная перегрузка, отслеживание веса и тренировок.

Работает как PWA — можно "Добавить на главный экран" на iPhone/Android и пользоваться как обычным приложением. Данные хранятся в localStorage браузера.

## Деплой на Vercel (5 минут)

### Шаг 1: Поставь приложение на GitHub

Если ещё нет аккаунта — зарегистрируйся на [github.com](https://github.com).

```bash
# В папке gym-coach/
git init
git add .
git commit -m "init gym coach"
```

Создай новый пустой репозиторий на GitHub (например, `gym-coach`). **Без README, без gitignore** — просто пустой.

Затем подключи и запушь:

```bash
git remote add origin https://github.com/ТВОЙ_НИК/gym-coach.git
git branch -M main
git push -u origin main
```

### Шаг 2: Залей на Vercel

1. Идёшь на [vercel.com](https://vercel.com) → **Sign up** через GitHub
2. На дашборде жмёшь **Add New → Project**
3. Выбираешь свой репозиторий `gym-coach` → **Import**
4. Ничего не меняй в настройках — Vite определяется автоматически (Framework Preset: Vite)
5. Жмёшь **Deploy**
6. Через ~1 минуту получаешь ссылку вида `https://gym-coach-abc123.vercel.app`

### Шаг 3: Открой на телефоне и добавь на домой экран

**iPhone (Safari):**
1. Открой ссылку из Vercel в Safari
2. Жми кнопку "Поделиться" (квадрат со стрелкой вверх внизу)
3. Прокрути вниз → **На экран «Домой»**
4. Назови как хочешь → Добавить

Теперь у тебя иконка на главном экране, открывается в полноэкранном режиме без адресной строки Safari.

**Android (Chrome):**
1. Открой ссылку в Chrome
2. Меню (три точки) → **Добавить на главный экран** (или Chrome сам предложит)
3. Подтверди

## Локальная разработка

Если хочешь что-то поменять локально:

```bash
npm install
npm run dev
```

Откроется на `http://localhost:5173`.

Чтобы обновить версию на Vercel — просто запушь изменения в GitHub, Vercel автоматически передеплоит.

## Важно про данные

- Данные хранятся в **localStorage браузера**, на каждом устройстве свои
- Если очистишь данные сайта в браузере — все тренировки удалятся
- В профиле есть кнопка **"Экспорт данных"** — регулярно делай бэкап
- Между iPhone и ноутбуком данные **не синхронизируются**

Если нужна синхронизация между устройствами — скажи, добавлю Supabase или аналогичное облачное хранилище (10–15 минут работы).

## Как изменить программу

База тренировок — в `src/App.jsx`:
- Объект `EXERCISES` — словарь упражнений с техникой, ошибками, советами
- Объект `PROGRAMS` — три тренировки (A/B/C) со ссылками на упражнения

Чтобы добавить упражнение — добавь запись в `EXERCISES`, потом сошлись на её ID из `PROGRAMS`.
